Dart es gratuito y open source

Compila en ARM y código de máquina x64 para dispositivos móviles, de escritorio y back-end. O compila a JavaScript para la web
Optimizado para UI
  • Maduro y completo async-await para interfaces de usuario que contienen código controlado por eventos, junto con concurrencia basada en aislamiento
  • Un lenguaje de programación optimizado para construir usuario interfaces de usuario con funciones como null safety, el operador de propagación (spread) para expandir colecciones e if de colección para personalización de la interfaz de usuario para cada plataforma
  • Un lenguaje de programación que es fácil de aprender, con una sintaxis familiar
Desarrollo productivo
  • Realiza cambios en tu código fuente de forma iterativa, utilizando hot reload para ver instantáneamente el efecto en la aplicación en ejecución
  • Escribe código usando un sistema de tipo flexible con rico análisis estático y potentes herramientas configurables
  • Haz profiling, logging, y debugging con el editor de código de tu elección
Rápido en todas las plataformas
  • Aplicaciones de compilación AOT a código de máquina nativo para inicio instantáneo
  • Dirígete a la Web con compiladores para JavaScript completos, maduros y rápidos
  • Ejecuta código backend compatible con tu aplicación, escrito con un único lenguaje de programación
